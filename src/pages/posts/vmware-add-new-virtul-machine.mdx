## Before starting
> [install VMware Workstation Pro](/posts/use-vmware-workstation-for-free)\
Download the`.iso`file for windows11([Download](https://www.microsoft.com/software-download/windows11)) or the system you want to install 

Select`File>New Virtual Machine...> Typical(suggest)`
![typical](/posts/vmware-add-new-virtul-machine/step1.png)
Select your`.iso`file for the system you want to install (or you can select it later)
![typical](/posts/vmware-add-new-virtul-machine/step2.png)
Select the system type of Virtual Machine
![typical](/posts/vmware-add-new-virtul-machine/step3.png)
Name your Virtual Machine and choose the path for it
![typical](/posts/vmware-add-new-virtul-machine/step4.png)
Set the disk size of virtual machine (I suggest the disk size must to biger then recommend size)
![typical](/posts/vmware-add-new-virtul-machine/step5.png)
![typical](/posts/vmware-add-new-virtul-machine/step6.png)
Then you can click `Edit vitual machine setting`to setting your virtual machine
![typical](/posts/vmware-add-new-virtul-machine/poweron.png)
![typical](/posts/vmware-add-new-virtul-machine/setting1.png)
![typical](/posts/vmware-add-new-virtul-machine/poweron.png)
